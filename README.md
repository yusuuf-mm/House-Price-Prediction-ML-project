# House Price Prediction

A simple Machine Learning project to predict California house prices using ML models.  
Includes FastAPI API and Docker deployment.

## Structure

- notebooks/ : Jupyter notebook for EDA and model training
- src/ : model training script & trained model
- app/ : FastAPI API
- Dockerfile : Docker deployment

## How to run

### 1. Run Notebook
```bash
jupyter notebook notebooks/house_price_notebook.ipynb
